╔═══════════════════════════════════════════════════════════════════════════════╗
║  CHAPTER 3: taskFirstScan Method Internal Structure (Template Method)      ║
╚═══════════════════════════════════════════════════════════════════════════════╝

  ╔══════════════════════════════════════════════════════════════╗
  ║  taskFirstScan<T>(request, region, currentSnap, outCache)    ║
  ╠══════════════════════════════════════════════════════════════╣
  ║                                                              ║
  ║  ① if cancel → return                                       ║
  ║                                                              ║
  ║  ② if (String / ByteArray)  ──► String/AOB Path            ║
  ║     │                                                        ║
  ║     │  foreach chunk (64KB + overlap):                       ║
  ║     │    currentSnap->readData(region.base + off, memBuf)    ║
  ║     │    │                                                    ║
  ║     │    ├── if String:  performStringSearch(memBuf, ...)    ║
  ║     │    │               └── SimdScanner::findFirstChar()    ║
  ║     │    │                    + memcmp                       ║
  ║     │    │                                                    ║
  ║     │    └── if AOB:  performAobSearch(memBuf, ...)          ║
  ║     │                 └── SimdScanner::findFirstChar()        ║
  ║     │                      + nibble-wise compare             ║
  ║     │                                                        ║
  ║     │    push matched → outCache                             ║
  ║     │                                                        ║
  ║  ③ if (Numeric Types) ──► Numeric Path (SIMD Optimized)    ║
  ║     │                                                        ║
  ║     │  v1 = request.params.value1                            ║
  ║     │  v2 = request.params.value2  (for Between)             ║
  ║     │  // Fill targetBuf with repeated v1 values            ║
  ║     │                                                        ║
  ║     │  foreach chunk (64KB):                                 ║
  ║     │    currentSnap->readData(region.base + off, memBuf)   ║
  ║     │    │                                                    ║
  ║     │    ├── UnknownInitial:                                  ║
  ║     │    │  Record all aligned addresses                      ║
  ║     │    │                                                    ║
  ║     │    ├── ExactValue / GreaterThan / LessThan:             ║
  ║     │    │  SimdScanner::scanMemoryBlockForMatches<T>(        ║
  ║     │    │    memBuf, targetBuf, toRead, base, align, op)     ║
  ║     │    │  ┌──────────────────────────────────────────┐     ║
  ║     │    │  │ AVX2 256-bit Register Comparison:        │     ║
  ║     │    │  │ int8:   32/cycle  loadu→cmpeq→movemask  │     ║
  ║     │    │  │ int16:  16/cycle                        │     ║
  ║     │    │  │ int32:  8/cycle                         │     ║
  ║     │    │  │ int64:  4/cycle                         │     ║
  ║     │    │  │ float:  8/cycle  cmp_ps→movemask_ps    │     ║
  ║     │    │  │ double: 4/cycle  cmp_pd→movemask_pd    │     ║
  ║     │    │  └──────────────────────────────────────────┘     ║
  ║     │    │                                                    ║
  ║     │    └── Between / Float Approximate:                     ║
  ║     │      Scalar loop: curVal >= v1 && curVal <= v2         ║
  ║     │                                                        ║
  ║     │    push matched → outCache                             ║
  ║     │                                                        ║
  ║     m_progress.fetch_add(1)                                  ║
  ╚══════════════════════════════════════════════════════════════╝


╔═══════════════════════════════════════════════════════════════════════════════╗
║  CHAPTER 4: taskNextScan + Snapshot Relationships (Multi-Snapshot Compare) ║
╚═══════════════════════════════════════════════════════════════════════════════╝

  ╔══════════════════════════════════════════════════════════════╗
  ║  taskNextScan<T>(req, oldBatch, currentSnap, prevSnap,       ║
  ║                  firstSnap, outCache)                         ║
  ╠══════════════════════════════════════════════════════════════╣
  ║                                                              ║
  ║  For each oldResult.address in oldBatch:                    ║
  ║                                                              ║
  ║    Current Snapshot ──► currentSnap->readValue(addr, curVal) ║
  ║                                                              ║
  ║    Old Snapshot    ──► prevSnap->readValue(addr, oldVal)     ║
  ║                                                              ║
  ║    First Snapshot  ──► firstSnap->readValue(addr, firstVal)  ║
  ║                                                              ║
  ║    Evaluate based on nextType:                              ║
  ║     ┌──────────────────────────────────────────────────┐     ║
  ║     │  Equal:              curVal == v1               │     ║
  ║     │  NotEqual:           curVal != v1               │     ║
  ║     │  Increased:          curVal > oldVal            │     ║
  ║     │  Decreased:          curVal < oldVal            │     ║
  ║     │  Changed:            curVal != oldVal           │     ║
  ║     │  Unchanged:          curVal == oldVal           │     ║
  ║     │  Between:            v1 <= curVal <= v2        │     ║
  ║     │  IncreasedBy:        curVal > oldVal + v1      │     ║
  ║     │  DecreasedBy:        curVal < oldVal - v1      │     ║
  ║     │  Compare_to_First:   curVal == firstVal        │     ║
  ║     └──────────────────────────────────────────────────┘     ║
  ║                                                              ║
  ║    if match → push to survivors → outCache                  ║
  ║                                                              ║
  ╚══════════════════════════════════════════════════════════════╝

╔═══════════════════════════════════════════════════════════════════════════════╗
║  CHAPTER 5: SIMD Acceleration Internal (AVX2 256-bit Vector Processing)    ║
╚═══════════════════════════════════════════════════════════════════════════════╝

   SimdScanner (Static Class, Pure Functions)
        │
        ├── scanMemoryBlockForMatches<T>()
        │       │
        │       │  Input: processMemory*, targetFilledBlock*, size, base, align, op
        │       ▼
        │   ┌──────────────────────────────────────────────┐
        │   │ Main Loop: offset = 0; offset+32 <= size;    │
        │   │             offset += 32                     │
        │   │                                              │
        │   │   ┌──────────┐    ┌──────────┐              │
        │   │   │ currentChunk│   │ targetChunk │            │
        │   │   │  (32 bytes) │   │  (32 bytes) │            │
        │   │   └─────┬──────┘   └──────┬───────┘            │
        │   │         │                 │                     │
        │   │         ▼                 ▼                     │
        │   │      ┌──────────────────────────┐              │
        │   │      │     AVX2 Compare Instr    │              │
        │   │      │                          │              │
        │   │      │  int8:  _mm256_cmpeq_epi8│              │
        │   │      │ int16: _mm256_cmpeq_epi16│              │
        │   │      │ int32: _mm256_cmpeq_epi32│              │
        │   │      │ int64: _mm256_cmpeq_epi64│              │
        │   │      │ float: _mm256_cmp_ps     │              │
        │   │      │double: _mm256_cmp_pd     │              │
        │   │      └────────────┬─────────────┘              │
        │   │                   │                             │
        │   │                   ▼                             │
        │   │      ┌──────────────────────────┐              │
        │   │      │  _mm256_movemask_epi8/ps/pd│              │
        │   │      │  → uint32_t matchBitmask  │              │
        │   │      └────────────┬─────────────┘              │
        │   │                   │                             │
        │   │         matchBitmask != 0?                     │
        │   │                   │                             │
        │   │                   ▼ (Yes)                       │
        │   │      ┌──────────────────────────┐              │
        │   │      │ Iterate each element     │              │
        │   │      │ within the SIMD block    │              │
        │   │      │ Extract matches → push   │              │
        │   │      └──────────────────────────┘              │
        │   └──────────────────────────────────────────────┘  │
        │                                                     │
        │   tail: Scalar fallback (< 32 bytes remaining)      │
        │                                                     │
        ├── findFirstChar()
        │       │
        │       │  Input: memory*, size, byteToFind
        │       ▼
        │   ┌──────────────────────────────────────────────┐
        │   │  broadcastByte = _mm256_set1_epi8(byteToFind)│
        │   │  foreach 32-byte chunk:                       │
        │   │    eq = _mm256_cmpeq_epi8(chunk, broadcast)   │
        │   │    mask = _mm256_movemask_epi8(eq)            │
        │   │    while mask:                                │
        │   │      _BitScanForward(&bit, mask)              │
        │   │      foundOffsets.push(offset + bit)          │
        │   │      mask &= ~(1 << bit)                      │
        │   └──────────────────────────────────────────────┘
        │
        └── Data Type → AVX2 Instruction Mapping Table:

            ╔═══════════╦══════════════════════════╦══════════════╗
            ║ Data Type ║ Compare Instruction      ║ Elements/Cycle║
            ╠═══════════╬══════════════════════════╬══════════════╣
            ║ int8      ║ _mm256_cmpeq_epi8        ║     32       ║
            ║ int16     ║ _mm256_cmpeq_epi16       ║     16       ║
            ║ int32     ║ _mm256_cmpeq_epi32       ║     8        ║
            ║ int64     ║ _mm256_cmpeq_epi64       ║     4        ║
            ║ float     ║ _mm256_cmp_ps            ║     8        ║
            ║ double    ║ _mm256_cmp_pd            ║     4        ║
            ╚═══════════╩══════════════════════════╩══════════════╝


╔═══════════════════════════════════════════════════════════════════════════════╗
║  CHAPTER 6: AdaptiveCachePool Architecture (Lock-Free Thread-Local Cache)  ║
╚═══════════════════════════════════════════════════════════════════════════════╝

   ┌─────────────────────────────────────────────────────────────────────┐
   │  AdaptiveCachePool<ScanResult>                                      │
   │  (Manages N AdaptiveCache sub-caches)                               │
   │  ┌───────────────────────────────────────────────────────────────┐  │
   │  │                                                               │  │
   │  │   Thread 1 ─── thread_local ──► AdaptiveCache #1              │  │
   │  │                                    │ push_back(data)          │  │
   │  │                                    │                          │  │
   │  │                                    ├── memBuf (vector<T>)     │  │
   │  │                                    │    until memory threshold │  │
   │  │                                    │                          │  │
   │  │                                    ├── Threshold exceeded?    │  │
   │  │                                    │    ↓                     │  │
   │  │                                    │  flushToDisk()           │  │
   │  │                                    │    ↓                     │  │
   │  │                                    │  .tmp file (spillover)   │  │
   │  │                                    │                          │  │
   │  │   Thread 2 ─── thread_local ──► AdaptiveCache #2              │  │
   │  │                                    │ ...                      │  │
   │  │                                    │                          │  │
   │  │   Thread N ─── thread_local ──► AdaptiveCache #N              │  │
   │  │                                    │ ...                      │  │
   │  │                                                               │  │
   │  │   Aggregated Read: readChunk(start, count)                     │  │
   │  │                ↓                                              │  │
   │  │    Iterate all sub-caches, skip out-of-range ones,             │  │
   │  │    read slice from matching cache and concatenate              │  │
   │  │                                                               │  │
   │ └───────────────────────────────────────────────────────────────┘  │
   │                                                                     │
   │  [Design Goal] Allow N threads to write concurrently               │
   │                with ZERO lock contention                            │
   └─────────────────────────────────────────────────────────────────────┘
